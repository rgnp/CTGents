# 开发路线图：终极目标 — 自进化 + AIGC

> 以当前 Agent 框架为基础，走向两个终极能力：
> **自进化（Self-Evolution）** — 给自己加想要的功能
> **AIGC（AI Generated Creativity）** — 产生创新的东西

---

## 🎯 愿景

```
自进化闭环已跑通：
  发现问题 → 问你"要修吗？" → 你说修 → 我分析+修改+验证
                                 ↑
                           install_plugin 已就绪
```

---

## 🚧 当前状态

### 已有能力

| 能力 | 状态 | 说人话 |
|------|:----:|--------|
| 会话管理 | ✅ | 保存/恢复/重命名/导出/列表 |
| 指令系统 | ✅ | 结构化注册、热加载、/help |
| MCP 协议 | ✅ | 连接任意 MCP 服务器 |
| 上下文缓存 | ✅ | 三段式 CacheContext |
| RAG 索引 | ✅ | 语义搜索项目代码 |
| 工具并行 | ✅ | 只读并行，写串行 |
| 工具去重 | ✅ | 同轮滑动窗口去重 |
| 安全模式 | ✅ | MANUAL / SAFE / AUTO |
| **自省 (/self)** | **✅** | 查看架构、工具、命令、插件全景 |
| **自跟踪 (/stats)** | **✅** | 工具调用统计（成功率、耗时、重复模式） |
| **主动性 (suggest)** | **✅** | 会话结束后自动扫描，发现问题主动开口 |
| **修复闭环** | **✅** | 发现问题 → 问你要不要修 → 你说修 → 我执行 |
| **插件热加载** | **✅** | `install_plugin` 可用，plugins/ 就绪 |

### 当前差距

**自进化闭环已跑通，但还差最后一步：从"你同意后才修"到"自主修复"。**
现在是我发现问题 → 问你 → 你说修 → 我修。
未来是：小问题我直接修，修完告诉你一声。

## Phase 3：AIGC（创造）

> 能写代码 ≠ 能创造。创造是"这东西不存在，我想出来的"。

### 3.1 创造新工具

- [ ] 根据自然语言需求写插件（已有 `install_plugin` 能力）
- [ ] 工具自动注册、自动生成测试
- [ ] 使用效果自动反馈优化

### 3.2 创造新交互

- [ ] 发现重复模式 → 提议新命令/向导
- [ ] 自动生成交互式流程简化复杂操作

### 3.3 创造新知识

- [ ] 从项目库提炼最佳实践 → 生成文档
- [ ] 从多次修复提炼模式 → 生成 check 规则
- [ ] 跨项目迁移经验

### 3.4 开放式创新

- [ ] "你有什么建议？"— 主动提改进方案
- [ ] 识别"可以做得更好"的地方
- [ ] 探索新技术并评估是否值得接入
