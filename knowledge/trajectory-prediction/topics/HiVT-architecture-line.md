# HiVT 架构线

> 记录 HiVT 及其衍生架构的演进、定位和实验配置。

---

## 为什么 HiVT 是理想的实验平台

1. **轻量** — HiVT-64 仅 0.69M 参数，训练快、迭代快
2. **开源** — 官方代码完整，PyTorch Lightning 实现，易于修改
3. **裸架构** — 没有辅助任务，没有花哨的额外 loss，干净
4. **经典基线** — 被大量论文引用/对比，实验结果有说服力

## 技术栈

| 组件 | 版本/详情 |
|---|---|
| Python | 3.8 |
| PyTorch | 1.8.0 |
| PyTorch Geometric | 1.7.2 |
| PyTorch Lightning | 1.5.2 |
| 数据集 | Argoverse 1.1 |
| GPU | RTX 2080 Ti（官方） |

## 实验配置要点

- HiVT-64 单卡训练 35-40 分钟（RTX 2080 Ti）
- 首次运行需预处理数据（~3.5 小时）
- 预训练 checkpoint 可作为后续添加辅助任务的初始化
