# 论文卡片目录

每篇论文一个 markdown 文件，格式统一，便于维护和 RAG 检索。

## 命名规则

`<年份>-<关键词>.md`

## 每张卡片包含

- 论文元信息（会议/年份/代码）
- 核心贡献（一句话）
- 辅助任务设计（类型/缺失能力C/损失设计/标签来源）
- 实验结论（数据集/提升幅度/消融）
- 局限性
- 与项目关系
- 待办事项
