# Trajectron++: Dynamically-Feasible Trajectory Forecasting with Heterogeneous Data

| 字段 | 内容 |
|:----|:-----|
| **会议** | CoRL 2020 (v1) / TPAMI 2021 (v2) |
| **任务** | 多智能体轨迹预测（车辆+行人+自行车） |
| **数据集** | nuScenes, ETH/UCY |

## 核心贡献

1. **图结构RNN**：将场景中的agent作为节点，构建动态图
2. **CVAE框架**：条件变分自动编码器生成多模态轨迹
3. **运动学可行**：输出轨迹满足车辆运动学约束（自行车模型）
4. **异构数据处理**：统一处理不同类别agent

## 为什么重要

Trajectron++ 是最早将**结构化场景建模+生成式预测+物理可行**统一起来的代表作。后续很多工作在此基础上改进。

## 和辅助任务的关系

- 使用了 **CVAE的KL散度** 作为隐式约束（不是显示辅助任务）
- 运动学约束是硬编码（不是可微loss）

## 空白机会

- 没有辅助任务层面的创新
- 运动学约束作为**可微 loss** 的尝试（TPK 后来做了）
