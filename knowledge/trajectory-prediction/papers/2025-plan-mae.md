## 核心贡献

**现有问题**：预测和规划依赖监督学习，优化指标对齐真值，但忽略了场景理解层面的好处。

**核心思路**：用Masked Autoencoder（MAE）做自监督预训练，融合3个重构任务。

## 辅助任务设计

**类型**: 重构式辅助（Template C）

**缺失能力 C**:
1. 空间关联理解 → 重构被mask的道路网络
2. 社会交互建模 → 重构被mask的agent轨迹
3. 目的意图理解 → 重构被mask的导航路线

**额外的子规划任务**:
4. Ego车辆的近端轨迹段预测（基于前一段进行条件预测）
   → 对齐车辆动力学和安全约束

## 训练流程

```
预训练阶段（无监督MAE）
  ├── 重建道路网络（学习空间拓扑）
  ├── 重建agent轨迹（学习社会交互）
  └── 重建导航路线（学习目的意图）
       ↓
下游微调阶段
  └── 联合生成预测+规划轨迹
```

## 实验结论（基于摘要）

- 预训练后的模型在预测和规划指标上均优于从头训练的版本
- 三个重建任务各自有贡献（消融实验验证）

## 和本项目的关系

- **直接相关**：Plan-MAE代表了"自监督预训练"范式在轨迹预测中的应用
- **区别**：Plan-MAE是做两阶段（pre-train → fine-tune）；我们关注的是单阶段（训练时加辅助loss）
- **互补**：我们的方法可以和Plan-MAE结合 → 先用MAE预训练，再用辅助loss微调
