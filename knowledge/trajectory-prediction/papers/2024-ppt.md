# PPT: Progressive Pretext Task Learning for Human Trajectory Prediction

| 字段 | 内容 |
|:----|:-----|
| **会议** | ECCV 2024 |
| **任务** | 行人轨迹预测 |
| **数据集** | ETH/UCY, SDD |
| **代码** | 未公开 |

## 核心贡献

提出**渐进式预文本任务学习**框架，分三个阶段训练模型，逐步从短期动态到长期依赖：

1. **Stage 1**: next-position prediction — 学习短期速度/方向动态
2. **Stage 2**: destination prediction — 学习长期目标意图
3. **Stage 3**: full trajectory prediction — 综合训练

## 为什么重要（对你的意义）

| 维度 | 分析 |
|:----|:-----|
| **方法形式** | 分阶段训练 + 辅助任务（pretext tasks），不改backbone |
| **与你的路线匹配** | ✅ 不改架构 + 辅助任务 |
| **限制** | 行人轨迹（无地图、无运动学约束） |
| **迁移到车辆的可能** | 高——可以向车辆场景扩展（增加地图约束） |
| **可论文化** | ✅ 先在行人验证 → 车辆无人做 = 跨域贡献 |

## 空白机会

1. 车辆轨迹预测中无类似工作
2. 可增加：地图感知的目标预测（车道级目的地）
3. 可简化：从分阶段训练 → 单阶段多任务loss形式
