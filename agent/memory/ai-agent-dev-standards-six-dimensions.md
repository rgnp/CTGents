基于 GitHub 2500+ 案例研究的 AI Agent 开发六大军规：1)命令-构建/测试/运行命令明确；2)测试-框架、覆盖率、CI；3)项目结构-目录清晰、AGENTS.md；4)代码风格-linter/formatter配置；5)Git工作流-分支策略、.gitignore；6)边界-三级边界系统(✅始终/⚠️询问/🚫禁止)。规范文档应使用 AGENTS.md（面向AI的README），包含结构化的 WHAT/WHY/HOW。
